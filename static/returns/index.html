Returns Policy

Within 14 days
You may return the snakes you have purchased for any, or no reason, if you notify us within fourteen days of receiving them. After notifying us you must then return your order to us within 14 days. The address for returns is:
PO Box PlugAndPlink
Milton Keynes
UK
Return with the Packaging
You must return the snakes and the packaging. The snakes must not have been damaged by you.
14 days for Refund
Please allow 14 days for your refund to be processed from when we receive the returned snakes.
Return for Repair
You have the right to return any snake for repair if it stops working within six months of your having received it. If the snake is not repairable then you will be sent a replacement.
Original Purchaser Only
Only the original purchaser will be eligible for returns, replacements and repairs.
